# Технология разработки программный приложений. Практическая работа №1

# Цель работы
Изучить основы работы с системой контроля версий GIT.

В соответствии с номером моего варианта (№ 9) реализовать следующие задачи:
  - Склонировать непустой удаленный репозиторий на локальную машину;
  - Создать новую ветку и выведите список всех веток;
  - Произвести 5 коммитов в новой ветке в разные файлы;
  - Выгрузить изменения в удаленный репозиторий;
  - Произвести revert 2-го и 4-го коммита в новой ветке;
  - Вывести в консоли различия между веткой master и новой веткой;
  - Слить новую ветку с master при помощи merge.
